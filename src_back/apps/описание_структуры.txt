Все приложухи в пакете apps, а общие настройки в src_back
В пакете apps:
    - analitics пакет, который содержит:
        - simple_analytics приложение для простой аналитики
        - complex_analytics приложение для сложной аналитики
    - profile_manage управление профилем, входом, регистрацией
    - survey_manage пакет, который содержит:
        - survey_base приложение для управления основой опроса (название, описание, тип опроса)
        - question_blocks приложение для управления блоками вопроса в опросе
        - answer_blocks приложение для управления блоками ответа в вопросе
        (я их разделил, потому что в models сильно завал был,
            по идее можно оперировать ими в одном приложении, мб общее потом
                ещё создать)
    - survey_passing приложение для прохождения / просмотра созданного опроса
